

# AttendancePanda 🐼

A simple, clean attendance calculator with no login required.

## Features

1. **Input Fields**
   - Total classes held
   - Classes attended
   - Minimum required attendance % (default: 75%)

2. **Live Calculation Display**
   - Current attendance percentage shown prominently
   - Color-coded: green if above required, red if below

3. **Smart Advice**
   - **If above required %**: Shows how many classes the user can skip and still stay above the minimum
   - **If below required %**: Shows how many consecutive classes the user must attend to reach the minimum

4. **Design**
   - Friendly panda-themed branding with "AttendancePanda" title
   - Clean, centered card layout
   - Responsive for mobile and desktop
   - Subtle animations on results
